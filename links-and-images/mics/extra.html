Character Sets

You can use UTF-8 to count in Turkish:

  1. bir
  2. iki
  3. üç
  4. dört
  5. beş

HTML entities

There are 3 reserved characters in HTML: <, > and &.
You should always use HTML entities for these three characters.

"Quotes"

If you’re (you're/you`re) into “web typography”, you’ll also find yourself using curly quotes quite a bit.
